  • HTML é uma linguagem de marcação, sendo o bloco de formação mais basico da web. É nele onde voce define a estrutura e o significado de cada conteúdo do documento.
  • Andando de mãos dadas ao HTML ou XML, o CSS define a estilização de cada conteúdo do documento ao qual ele foi aplicado. É com ele que você deixa o conteúdo do seu site mais atrativo.

  • JavaScript é uma linguagem de programação de alto nivel. Junto ao HTML e CSS, o JavaScript é uma das principais tecnologias web, servindo para fazer páginas interativas ao usuário.
